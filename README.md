# Classic_Clash

## Goal
- Defeat all of the monsters on the map before you run out of health

## Controls
- Move : WASD
- Attack : Left-Mouse-Button
- Quit : Escape

## Details 
- Built with C++ using the raylib library
- Completed 05/24/22 from GaneDev.TV tutorial
